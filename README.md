## راهنمای نصب و استفاده (Windows)

### پیش‌نیازها
      - **ویندوز 10 یا 11 **
      -**حساب گوگل** 
      -**فیلترشکن (در صورت نیاز)** 

### گام‌به‌گام

1. **دانلود فایل ZIP**  
   - لینک دانلود را باز کنید و فایل ZIP را ذخیره کنید.  
2. **ایجاد کلید 16 رقمی**  
   - رشته‌ی تصادفی شامل حروف (Büyük/küçük)، اعداد، و نمادها بسازید (مثلاً `A7B9-C0D3-EF12-34GH`). طول دقیقاً ۱۶ کاراکتر باشد.  
3. **باز کردن کد Apps Script**  
   - پوشه استخراج‌شده → `apps_script` → فایل `code.gs`.  
4. **وارد کردن کلید**  
   - در خط 14، مقدار `const AUTH_KEY = "کد_شما";` را وارد کنید.  
5. **کپی کل محتویات فایل**  
   - (Ctrl+A, Ctrl+C) همه‌ی متن را کپی کنید.  
6. **ورود به Google Apps Script**  
   - مرورگر → `https://script.google.com`. با حساب گوگل وارد شوید.  
7. **ایجاد اسکریپت جدید**  
   - روی **New Script** کلیک کنید، منتظر بارگذاری صفحه‌ی کد باشید.  
8. **چسباندن کد**  
   - متن پیش‌فرض را پاک کرده و محتویات کپی‌شده (Ctrl+V) را بچسبانید.  
9. **انتشار Web App**  
   - بر روی **Deploy → New Deploy** کلیک کنید؛ در پنجره‌ی پاپ‌آپ گزینه‌ی **Web app** را انتخاب کنید.  
10. **تنظیمات انتشار**  
    - `Execute as` = *Me (email)*  
    - `Who has access to` = *Anyone*  
    - بر روی **Deploy** کلیک کنید؛ در صورت نمایش هشدار، **Advanced → Proceed** را بزنید.  
11. **ذخیره Deployment ID و URL**  
    - پس از انتشار، دو مقدار (Deployment ID, URL) نمایش داده می‌شوند؛ آن‌ها را کپی کرده و ذخیره کنید.  
12. **اجرای اسکریپت پایتون**  
    - به پوشه‌ی استخراج‌شده → فایل `start.bat` را دوبار کلیک کنید. CMD باز می‌شود، وابستگی‌های پایتون نصب می‌شوند.  
13. **وارد کردن مقادیر در CMD**  
    1. Auth key (کلید ۱۶ رقمی)  
    2. Deployment ID  
    3. `y` برای تأیید تغییرات LAN  
    4. `127.0.0.1` (آدرس لوکال هاب)  
    5. `8085` (پورت پروکسی HTTP)  
    6. `1080` (پورت SOCKS)  
14. **تایید گواهینامه**  
    - در پنجره‌ی امنیتی، گزینه‌ی `YES` را بزنید.  
15. **تنظیمات Proxy ویندوز**  
    - Settings → Network & Internet → Proxy → Manual proxy setup:  
      - Address: `127.0.0.1`  
      - Port: `8085`  
      - Use a proxy server = On  
      - Save.  
16. **تکمیل نصب**  
    - تمام مراحل انجام شد؛ مرورگرها حالا از پروکسی محلی استفاده می‌کنند و دسترسی به وب‌سایت‌های مسدود‌شده برقرار است.
