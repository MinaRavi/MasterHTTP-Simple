# روش اول برای ویندوز

## 1. دانلود فایل
اول فایل زیپ زیر را از این لینک دانلود کنید.
https://github.com/MinaRavi/MasterHTTP-Simple/blob/main/MasterHTTP-Simple.rar

---

## 2. ساخت Auth Key
یک کد 16 رقمی با حروف، اعداد و اشکال بسازید و یک جا یادداشت کنید.

---

## 3. ویرایش فایل `code.gs`
- فایل زیپ را استخراج کنید.
- وارد پوشه `apps_script` شوید.
- فایل `code.gs` را باز کنید.

در خط 14، `"const AUTH_KEY"` را پیدا کنید و کد 16 رقمی که در مرحله 2 ساختید را داخل `""` قرار دهید.

---

## 4. کپی کردن کدها
تمام کدهای داخل فایل Code.gs را کپی کنید.

Ctrl + A
Ctrl + C


---

## 5. ورود به Google Apps Script
وارد آدرس زیر شوید:

```text
script.google.com
```

> برای بعضی‌ها این آدرس بدون فیلترشکن باز می‌شود؛ اگر نه، فقط یک‌بار نیاز است با فیلترشکن وارد شوید.

---

## 6. ساخت Script جدید
- با یکی از حساب‌های گوگل‌تان وارد شوید.
- ترجیحاً از یک ایمیل کم‌اهمیت استفاده کنید.
- از سمت چپ روی `New Script` کلیک کنید.
- صبر کنید تا صفحه کدنویسی باز شود.

> ممکن است لود شدن صفحه کمی طول بکشد.

---

## 7. جایگزین کردن کدها
- کدهای پیش‌فرض صفحه را پاک کنید.
- کدهایی را که در مرحله 4 کپی کردید (Code.gs) جایگزین کنید.

---

## 8.اعمال کردن پروژه
از بالا سمت راست:

```text
Deploy → New Deployment
```

را انتخاب کنید.

سپس:
- روی چرخ‌دنده سمت چپ کلیک کنید.
- گزینه `Web app` را بزنید.

---

## 9. تنظیم دسترسی
مقادیر زیر را انتخاب کنید:

```text
Execute as → Me (email)
Who has access to → Anyone
```

سپس روی `Deploy` کلیک کنید.

---

## 10. تایید دسترسی‌ها
ممکن است:
- لود شدن این مرحله کمی طول بکشد.
- صفحه اخطار امنیتی باز شود.

در این صورت:
- روی `Advanced` کلیک کنید.
- دسترسی لازم را تایید کنید.

> اخطار امنیتی فقط به دلیل استفاده از متد HTTP است.

---

## 11. ذخیره اطلاعات Deploy
بعد از Deploy شدن:
- `Deployment ID`
- `URL`

به شما نمایش داده می‌شود.

هر دو را کپی کرده و ذخیره کنید.

---

## 12. اجرای فایل Start
به پوشه‌ای که در مرحله اول استخراج کردید برگردید و فایل زیر را اجرا کنید:

```text
start.bat
```

---

## 13. نصب خودکار پیش‌نیازها
یک پنجره CMD باز می‌شود و:
- Python
- پیش‌نیازهای اجرایی

به‌صورت خودکار دانلود و نصب می‌شوند.

---

## 14. وارد کردن اطلاعات
به‌ترتیب مقادیر زیر از شما پرسیده می‌شود:

### Auth Key
همان کد 16 رقمی مرحله 2 را وارد کنید.

### Deployment ID
مقداری که در مرحله 11 ذخیره کردید را وارد کنید.

### تایید تغییر تنظیمات LAN
حرف زیر را وارد کنید:

```text
y
```

### Proxy IP
```text
127.0.0.1
```

### Proxy Port
```text
8085
```

### Socks Port
```text
1080
```

---

## 15. نصب گواهینامه
اخطار امنیتی نصب گواهینامه باز می‌شود.

روی:

```text
YES
```

کلیک کنید.

---

## 16. تنظیم Proxy ویندوز
وارد مسیر زیر شوید:

```text
Settings → Network & Internet → Proxy
```

در قسمت `Manual Proxy Setup`:

| گزینه | مقدار |
|---|---|
| Proxy IP Address | `127.0.0.1` |
| Port | `8085` |

سپس:
- گزینه `Use a proxy server` را روی `On` بگذارید.
- تنظیمات را ذخیره کنید.
