# MasterHTTP-Simple
برای ویندوز 
1-	اول فایل زیپ زیر رو از این لینک دانلود کنید.
2-	یه کد 16 رقمی با حروف و اعداد و اشکال بسازید و یه جا بنویسیدش .
3-	فایل زیپ رو استخراج کنید و از وارد پوشه apps_script  بشید و فایل code.gs  رو باز کنید.
4-	توی خط 14 ام “const AUTH_KEY” رو پیدا کنید و کد 16 رقمی که در مرحله 2 ساختید رو داخل "" قرار بدید.
5-	تمام کد های داخل فایل رو کپی کنید . ((ctrl+A, ctrl+C
6-	وارد آدرس script.google.com بشید. (برای بعضی این ادرس بدون فیلتر شکن باز میشه – اگر نه نیازه که فقط یکبار با فیلتر شکن وارد شید.)
7-	با یک حساب گوگلتون لاگین کنید (اگر از قبل لاگین نیستید! / ترجیحا یه ایمیل کم اهمیت رو انتخاب کنید.)
8-	از سمت چپ روی گزینه New Script  کلیک کنید و صبر کنید تا صفحه کد براتون باز شه (ممکنه لودش طول بکشه پس صبور باشید.)
9-	توی صفحه باز شده کد هایی رو که در مرحله 5 کپی کردید رو به جای کد های جدید اضافه کنید ( کد های پیشفرض در صفحه رو پاک کنید)
10-	 و روی بار بالا سمت راست روی Deploy  سپس New Deploy  بزنید.
11-	 روی چرخ دنده سمت چپ کلیک کنید و Web app  رو بزنید.
12-	مقادیر Execute as -> me (email) و who has access to -> Anyone رو انتخاب کنید و روی Deploy کلیک کنید .
13-	 لود شدن این مرحله ممکنه طول بکشه و ازتون اجازه دسترسی بخواد و صفحه نا امن بودن رو باز کنه که شما روی advanced  کلیک میکنید و اجازه دسترسی بهش میدید ( اخطار امنیت فقط به دلیل استفاده از متد HTTP  است پس نگران نباشید )
14-	صفحه جدیدی باز میشود که مقادیر Deployment ID  و URL به شما نشان داده میشود. 
15-	هر دو مقدار را کپی و در جایی ذخیره کنید. 
16-	حالا وارد پوشه زیپی که در مرحله اول استخراج کردیم میشیم و فایل start.bat رو اجرا میکنیم.
17-	در این مرحله یک صفحه CMD باز شده و پایتون و پیش نیاز های اجرایی به طور خود کار دانلود و نصب میشود.
18-	 سپس مقادیری که از قبل ذخیره کرده ایم به ترتیب پرسیده میشود
19-	Auth key  یا همان کد 16 رقمی مرحله 2 را وارد میکنیم.
20-	Deployment ID را که از مرحله 15 ذخیره کردیم وارد میکنیم.
21-	حرف y  را وارد میکنیم. )قبول کردن تغییر تنظیمات LAN)
22-	آدرس 127.0.0.1 را وارد میکنیم.
23-	پورت پروکسی 8085 را وارد میکنیم.
24-	پورت ساکس 1080 را وارد میکنیم.
25-	 احطار امنیتی برا نصب گواهینامه باز میشود که YES  را میزنیم.
26-	وارد تنظیمات ویندوز شده Network & Internet سپس Proxy 
27-	در قسمت Manual Proxy Setup  کادر proxy IP addres -> 127.0.0.1 و Port -> 8085 را پر میکنیم و use a proxy server ->On و سیو میکنیم.
28-	تبریک ! شما دسترسی به youtube telegram  Instagram و بساری از سایت های بین المللی را دارید!
